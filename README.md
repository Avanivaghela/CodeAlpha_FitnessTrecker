# Fitness Tracker App

A Java-based Fitness Tracker App developed for the CodeAlpha Internship.

## Features
- Add workout details
- Track steps and calories
- Record workout time
- Daily dashboard
- Weekly summary
- SQLite database
- Workout history
- Progress bar

## Technologies Used
- Java
- Android Studio
- SQLite
- XML
